第一台万元机

下定决心装一台深度学习工作站

搞深度学习，没有一台自己的机器，实在是不方便。

说搞就搞，本人对于搞有用的东西，从来不犹豫。

查看当前的主流配置

师弟那听来的装机大师，百度一下发现了下面这个网站。

攒机之家
http://zgcdiy.com/diy.html

包括价格，购买量，参数，都有了，可以在这里虚拟的试着攒一台。

并且了解到许多信息，对当前主流配置有了一定的把握。

了解到的知识：

  • cpu插槽 LGA1151
  • cpu散片与盒装的区别

最终决定购买的清单：

  • cpu i7 7700K
  • 华硕 Z170AR
  • 金士顿 Fuzzy 16G
  • WD 1TB 7200
  • GPU 1080ti
  • 电源 650W-800W
  • 散热器 九州风神 玄冰四百
  • 机箱

上京东，查找配置

了解到大概定位后，还得上京东，看销量，评价，价格

经过一番浏览之后，决定一下配置：

  • cpu i7 7700k 2000
  • 金士顿 Fuzzy 16G 1000
  • 华硕 Z270A 1200
  • GPU 1080ti 技嘉大雕 evga 6000+
  • 电源 爱国者800W 550
  • 机箱 爱国者 250
  • 散热器 九州风神玄冰四百 100
  • 硬盘 WD 1TB 7200 280
  • SSD 三星 120（原来笔记本上拆下来的）

一定要买自营，一定要在旗舰店买，不要买套装。

装机

cpu撞到主板上，很简单

装上cpu散热器

这是最复杂的一步了。

  • 一定要去网上看教程。
  • 先看好风扇的方向，一定要先比划好方向。
  • 这款散热器自带的硅胶很烂，很硬，还好我有原来留下的。
  • 一定要把散热器接触面擦干净，可以用化妆棉，沾点酒精
  • 打磨一下，这个散热器接触面不光滑，手都能划破，cpu不得划烂了。
  • 硅胶涂抹的要点是，要薄，要均匀，对齐之后稍微转两下。
  • 卡好，有点紧，别把主板压烂了。

主板装到机箱里

  • 先把9个柱脚排列好。
  • 注意有一个机箱挡板，要装好
  • 对齐拧上

硬盘

SSD和HDD都装到机箱里，有专门的位置，很精巧。

布线

机箱主板连接线，一定要看主板说明书，找准每个柱脚的位置。

  • usb
  • HD-audio(AAFP)
  • power_sw
  • reset_sw
  • power_led
  • hdd_led

风扇连接线

注意一定要住到主板上主风扇连接口的位置–CPU_FAN
否则BIOS会提示 CPU FAN SPEED ERROR，测试不到cpu风扇转速。
这在大型服务器中是一个很严重的错误。
